Mini Kurs HTML 5 i CSS 3 - Stanisław Szymczykowski 3G

Witaj w mini kursie HTML5 i CSS3! Ten kurs wprowadzi Cię w podstawy tworzenia stron internetowych przy użyciu najnowszych standardów HTML5 i CSS3. Odkryjemy, jak używać HTML5 do definiowania struktury i zawartości strony, a także jak wykorzystać CSS3 do nadawania jej atrakcyjnego wyglądu. Przygotuj się na praktyczne ćwiczenia i naukę, która pozwoli Ci zacząć tworzyć swoje własne strony internetowe!

Czym jest HTML?

HTML (HyperText Markup Language) to język znaczników używany do tworzenia stron internetowych. Jest to podstawowy budulec, na którym opiera się struktura i zawartość witryn internetowych. HTML składa się z zestawu znaczników, które definiują elementy strony, takie jak nagłówki, paragrafy, listy, linki, obrazy i wiele innych. HTML jest językiem opisowym, który informuje przeglądarkę internetową o sposobie prezentacji i organizacji treści na stronie. Znaczniki HTML są otwierane za pomocą nawiasów ostrych (<>) i mogą mieć atrybuty, które określają dodatkowe właściwości danego elementu. Na przykład, znacznik `` jest używany do wstawiania obrazów na stronie i może mieć atrybuty takie jak "src" (określający źródło obrazu) i "alt" (określający alternatywny tekst, który jest wyświetlany, gdy obraz nie może zostać załadowany). Przeglądarki internetowe interpretują kod HTML i renderują go jako widoczną stronę dla użytkownika. Oprócz samego tekstu i multimediów, HTML umożliwia również tworzenie linków do innych stron, tworzenie formularzy, tabel i innych elementów interaktywnych. Wraz z CSS (Cascading Style Sheets) i JavaScript, HTML stanowi podstawową trójkę technologii używanych do tworzenia dynamicznych i interaktywnych stron internetowych. CSS odpowiada za stylowanie strony, podczas gdy JavaScript umożliwia dodawanie interaktywności i reakcji na działania użytkownika. W skrócie, HTML jest językiem znaczników używanym do tworzenia struktur i treści stron internetowych, które są wyświetlane w przeglądarkach internetowych.

Czym jest CSS?

CSS (Cascading Style Sheets) to język używany do opisu wyglądu i stylizacji dokumentów HTML. Jest to technologia, która umożliwia tworzenie atrakcyjnych i spójnych projektów graficznych dla stron internetowych. CSS oddziela prezentację strony od jej struktury. Za pomocą CSS możemy definiować różne właściwości, takie jak kolory, czcionki, marginesy, paddingi, tła, rozmiary elementów itp. Wprowadza również możliwość tworzenia efektów animacyjnych, transformacji i przejść między stanami elementów strony. CSS działa na zasadzie kaskadowości, co oznacza, że można zdefiniować reguły stylizacji na różnych poziomach. Style mogą być definiowane zarówno wewnątrz dokumentu HTML (wewnątrz tagu `<style >`), jak i w zewnętrznym pliku CSS, który jest dołączany do strony. CSS pozwala również na tworzenie selektorów, które umożliwiają precyzyjne określanie, które elementy strony mają być stylizowane. Można wybierać elementy na podstawie ich typu, identyfikatora, klasy, hierarchii i innych atrybutów. W połączeniu z HTML i JavaScript, CSS stanowi trzon technologii używanych do tworzenia stron internetowych. HTML odpowiada za strukturę strony, CSS za jej stylizację, a JavaScript za interaktywność i dynamikę. W skrócie, CSS to język używany do określania wyglądu i stylizacji stron internetowych. Pozwala na separację prezentacji od struktury dokumentu HTML, co umożliwia tworzenie atrakcyjnych i spójnych projektów graficznych.
